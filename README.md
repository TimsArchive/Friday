# Friday
### Terminal Desktop Companion

> [!IMPORTANT]
> This is a prototype/proof of concept and is not entierly bug-free.
> In short; life got in the way.

## +--------------------- [ Hello ] ---------------------+

Friday is a **general use companion** for windows PCs that runs in the terminal, written in MS Batch.

Friday is highly **modular and customizable** and is created with scaling in mind so
feel free to modify & adjust the program to your heart's content!

**Want to view the source code?**
- To view the source code of any .bat file; rightclick -> Edit
    - This opens it in a text editor, like notepad.
- Or view it on GitHub.

Thank you for downloading and trying out my program! :)

## +--------------------- [ Installing ] ---------------------+

1. Extract the zip file to the Desktop.
    - *(C:\Users\\\<USER\>\Desktop)*
    - *'README.md' can be removed or relocated to any location*
2. Run the "Friday_Installer.bat".
3. Choose "Install Friday".

> [!IMPORTANT]
> - The program will automatically start once the installation is complete.

> [!TIP]
> - View the "Friday Tree" picture for easier folder navigation.
> - The program can be started using the 'Friday' desktop shortcut.


## +--------------------- [ Uninstalling ] ---------------------+

**-- Option 1 --**

1. Start Friday.
    - *(Or Friday Utility.)*
2. Go to Menu pages
    - *(Option 9)* -> Friday Utility -> *(Option 2)*.
3. Select "Uninstall Friday"
    - *(Option 3)*.

**-- Option 2 --**

1. Go to the installation directory.
    - *(By default-> C:\Users\\\<USER\>\Documents)*
2. Remove the "Friday_Root" directory.


> [!IMPORTANT]
> - Both methods will completely remove Friday.
> - This includes: files created, stored and used by Friday.
> - If you want to keep a certain file *(like a note)*, make a copy of it somewhere else.


## +--------------------- [ Story ] ---------------------+

This program started as a very small file of roughly ~50 rows called "Tarvis".
This was a parody name of the AI companion "Jarvis" from Iron man.
The goal of that program was to start certain programs with specifc
settings activated *(like the replay function in OBS)*. But after
expanding it a bit more I realized it could be used for so much more.

The program was intentionally kept in the language MS Batch to make it
easier to handle and call certain files and locations on the local computer.
The language Batch is also native to basically all windows machines making it
largely compatible and very easy to install. Another great feature with batch is that all
.bat files and their code are open source, simply right-click and choose 
open in notepad *(or any other text editor)*.
